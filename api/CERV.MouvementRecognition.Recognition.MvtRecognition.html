Class MvtRecognition

The MvtRecognition class. Contains all the methods to detect and analyze movements.

  • initiateValues: Initiate the different values of the class.
  • CheckIfMvtIsRight: Compare the movements made by the user with the one he should reproduce.
  • checkBeginningMvt: Check if the user is doing the beginning of a movement.
  • changeColorUICharacter: Change the color of the character attached to the UI.
  • launchComparison: Exist in two versions, used to compare the user position and a frame of a bvh file.
