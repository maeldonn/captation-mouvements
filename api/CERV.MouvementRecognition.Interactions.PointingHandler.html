Class PointingHandler

The pointingHandler class. Contains almost all the methods to handle the pointing system. To interact with the canvas some of them are in the mocapInputModule script that is attached to the EventSystem.

  • UpdateUserInputs: Update all that is related to the user input.
  • PlaySoundWhilePointing: Play a sound while pointing, and update the statePointing variable.
  • HandleClicks: Handle the visual effects of the clicks.
  • InitPointingHandler: Initialize the values used by the pointingHandler script.
  • DrawLineUsedToInteract: Draws the line used to interact with the menu.
  • GetConfirmState: Return the state of the stateConfirm variable.
